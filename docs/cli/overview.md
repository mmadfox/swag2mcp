# CLI Commands

swag2mcp provides 13 commands for management.

## Help

```bash
swag2mcp --help
swag2mcp <command> --help
```

## Command List

| Command | Description |
|---------|-------------|
| `init` | Initialize workspace |
| `add` | Add a spec |
| `delete` | Delete a spec or collection |
| `ls` | List specs and collections |
| `run` | Start TUI |
| `validate` | Validate configuration |
| `clean` | Clear cache |
| `update` | Update a spec |
| `mcp` | Start MCP server |
| `version` | Show version |
| `info` | Show system info |
| `import` | Import workspace |
| `export` | Export workspace |

## Global Flags

| Flag | Description |
|------|-------------|
| `--version` | Show version |
| `--help` | Show help |

Verbindet OpenAPI/Swagger/Postman-API-Spezifikationen mit LLM-Agenten über das Model Context Protocol (MCP).

## Ihre API spricht LLM

Eine einzige Konfigurationszeile verwandelt jede OpenAPI/Swagger/Postman-Datei in einen MCP-Server. LLM-Agenten entdecken, inspizieren und rufen Ihre APIs auf — null Integrationscode.

<img src="/architecture.svg" width="700" alt="swag2mcp-Architektur">

## Schluss mit Wrappern

Jedes Mal, wenn Sie eine neue API mit einem LLM verbinden, schreiben Sie denselben Boilerplate-Code: Spezifikationsparsing, Authentifizierung, Fehlerbehandlung, Ratenbegrenzung. swag2mcp erledigt das für Sie — 19 vorgefertigte MCP-Tools.

## Wer braucht das?

| Rolle | Warum |
|-------|-------|
| **AI-Agent-Entwickler** | Jede API in 2 Minuten verbinden, nicht 2 Tage |
| **MCP-Ingenieur** | Kein Handler-Code — einfach auf eine Spezifikation verweisen |
| **Architekt** | Einheitliche API-Integrationsschicht für alle LLMs im Unternehmen |
| **Datenanalyst** | APIs per natürlicher Sprache nutzen, ohne Programmierung |
| **DevOps / SRE** | Überwachung und Automatisierung durch LLM ohne zusätzliche Dienste |
| **Integrator** | 9 Authentifizierungsmethoden sofort einsatzbereit — Basic bis OAuth2 bis HMAC |
| **QA-Ingenieur** | Mock-Server für isolierte Tests ohne echte APIs |
| **Produktmanager** | Schnelle KI-Feature-Prototypen ohne Backend-Arbeit |
| **und viele weitere** | |
